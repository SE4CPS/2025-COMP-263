# Lab 5 Starter Resources

This folder provides:
- `sql_sensors.json` — small sensor lookup table
- `sql_readings_100rows.json` — 100-row readings table
- `lab5_starter.js` — Node.js starter (map/filter/reduce ONLY), with TODO-style structure built-in

## Run
```bash
node lab5_starter.js
```
